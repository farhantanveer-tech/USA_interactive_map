How regions are counted

This version uses a simple classroom-friendly split.

South: AL, AR, DE, FL, GA, KY, LA, MD, MS, NC, OK, SC, TN, TX, VA, WV, DC
North: all remaining states in the lower 48 plus AK and HI.

You can edit the South list inside the HTML file if your class uses a different definition.
